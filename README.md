# The Infinite Canopy

A modern, premium landing page about the universe and its wonders.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/import/git?s=https://github.com/aegish-zagi/universe-landing)

## Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Language**: TypeScript

## Sections

1. **Hero — The Infinite Canopy**: Full-bleed nebula with slow zoom, staggered type fade-in, and CTA
2. **Stellar Nurseries**: High-def star-forming region with parallax, split layout, and shimmer-in headline
3. **The Geometry of Spacetime**: Atmospheric section on light, gravity, and rotating ring visual
4. **Look Up — Final CTA**: Clean, focused invitation with cinematic typography

## Deploy

Click the "Deploy with Vercel" button above, or:

```bash
npm install
npm run dev
```

## Manual Vercel Deployment

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in project root
3. Follow the prompts
